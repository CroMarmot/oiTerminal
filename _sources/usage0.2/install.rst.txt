Install
=======

.. _installation:

Installation
------------

.. code-block:: console

    $ # install dependency
    $ git clone https://github.com/CroMarmot/yxr-atcoder-core.git
    $ cd yxr-atcoder-core
    $ pip3 install -e . --config-settings editable_mode=compat
    $ cd ..
    $
    $ git clone https://github.com/CroMarmot/oiTerminal -b He
    $ cd oiTerminal
    $ pip3 install -e .

Enable tab auto completion
--------------------------

.. code-block:: console

    $ source ./auto-completion/ot-auto-completion.sh

You can add this command in ``~/.bashrc`` for good better experience
