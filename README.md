## v0.1

### Install Dependency

```bash
git clone https://github.com/CroMarmot/yxr-atcoder-core.git
cd yxr-atcoder-core
pip3 install -e .
```

#### Enable tab auto completion

```bash
./auto-completion/gen-ot-auto-completion.sh
source /tmp/ot-auto-completion.sh
```
